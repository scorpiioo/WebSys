CSS Properties & Description

# Property Description
1 width Specifies the width of the element/div it is applied to. Eg: 80%, 400 px. If it is specified in %, then it is relative to the browser space. It is fixed in pixels.
2 height Specifies the height of the element/div it is applied to. Eg: 80%, 400 px. If it is specified in %, then it is relative to the browser space. It is fixed in pixels.
3 padding Specifies the distance between the border of the element/div and it's content. padding-left, padding-bottom, padding-right, padding-top can also be used.
4 margin Specifies the distance between the border of the element/div and the next/neighbouring div/element. margin-left, margin-bottom, margin-right, margin-top can also be used. Eg, 10%, 10px
5 border-radius It is used to round-en the borders of a div/element. Eg: 8Px, 8%
6 background-color It specifies the background color of the div/elemnt it is applied to. It can be given by some predefined color name, rgb value or hex value.
7 color It specifies the color of the text it is applied to. It can be given by some predefined color name, rgb value or hex value.
8 max-width Specifies the maximum width of the element/div it is applied to. It overrides the width specified.
9 overflow It specifies where and how the extra content is displayed when it doesn't fit it's parent div/element. Values: scroll, hidden, auto, inherit
10 font-family It specifies the font for the text it is applied to. Eg: serif, arial etc
11 font-size It specifies the size of the text it is applied to. Eg: 30px
12 float It specifies whether the element/div it is applied to should float or not and how if it floats. Values: none,left,right
13 text-align It specifies the horizontal(right.left/center) alignment of the text in the element/div which contains it.
14 font-weight It specifies the thickness of the text it is applied to. Values:normal, bold, light,bolder.
15 font-weight It specifies the thickness of the text it is applied to. Values:normal, bold, light,bolder.
16 display It specifies how the box used for a div/element is displayed. Values: inline, block, none etc.
17 text-decoration It specifies the decoration added to the text it is applied to. Values: underline, initial, none etc.
18 text-decoration It specifies the decoration added to the text it is applied to. Values: underline, initial, none etc.
